# Networking Labs

Folder ini berisi latihan dasar networking di Linux.

## Isi

- bind-address/  
  Latihan perbedaan 127.0.0.1 dan 0.0.0.0 menggunakan Node.js.

- python-http/  
  Latihan HTTP server sederhana menggunakan Python.

## Konsep yang dipelajari

- localhost
- 127.0.0.1
- 0.0.0.0
- IP LAN
- port
- bind address
- curl
- ss
- firewall/UFW
