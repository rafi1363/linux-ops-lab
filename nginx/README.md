# NGINX Labs

Folder ini berisi latihan NGINX dan reverse proxy.

## Isi

- nginx-reverse-proxy/  
  Lab reverse proxy dari NGINX port 80 ke Node.js 127.0.0.1:3000.

- config-backup/  
  Backup konfigurasi NGINX dari /etc/nginx/sites-available/.

## Konsep yang dipelajari

- web server
- reverse proxy
- proxy_pass
- public-facing service
- private backend
- access log
- error log
- attack surface minimization
